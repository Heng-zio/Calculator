# 🧮 Simple Calculator (C++)

A simple command-line calculator built with C++. It performs basic arithmetic operations using separate functions and a `switch` statement.

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🟰 Modulus (using integer conversion)

## 📂 Project Structure

```
calculator.cpp
README.md
```

## 🚀 How to Run

### Compile

```bash
g++ calculator.cpp -o calculator
```

### Execute

**Linux/macOS**
```bash
./calculator
```

**Windows**
```bash
calculator.exe
```

## 💻 Example

```
Enter your choice
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Modulus

Choice: 3
Enter two numbers: 8 6

Result: 48
```
